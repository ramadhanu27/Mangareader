## ✨ Overview

This repository contains the complete package to run a modern Manhwa, Manga, and Manhua reader website on WordPress.

1. **Flavor Flavor Theme (`flavor-flavor/`)**: A fully-featured WordPress theme built specifically for Manhwa/Manga reader websites. It features a built-in CMS, chapter management, fully customizable vertical scrolling reader, bookmark system, and much more.
2. **Manhwa Scraper Plugin (`manhwa-scraper/`)**: An optional companion plugin that works seamlessly with the theme to automatically scrape manhwa content.

## 🚀 Key Features

- **Manhwa CMS (Built-in)**: Add, edit, and delete Manhwas completely within the WordPress Admin. No messy third-party post types plugins needed.
- **Chapter Management**: Add chapters with image URLs, drag & drop sorting.
- **Reading Tools**: Vertical Scroll Reader, Reading History tracking, custom Bookmark System (via localStorage).
- **Responsive**: Mobile-first design that looks beautiful on desktop, tablet, and smartphone.
- **Color Customizable**: Easily change theme Primary and Secondary colors to adapt to your brand.
- **Dark / Light Mode**: Beautiful Dark Mode switch.
- **Interactions**: Built-in Custom Comments and Post Reactions (Like, Funny, Nice, Sad, Angry) without relying on heavy plugins.
- **Monetization Ready**: Strategically placed Ads Management Slots included natively.

For a comprehensive guide, feature list, and installation tutorial for the **Theme**, please see the [Flavor-Flavor Theme Documentation](flavor-flavor/README.md).

For setting up the **Scraper**, see the plugin folder `manhwa-scraper/`.

## 🛠️ Installation Guide

### 1️⃣ Installing the Theme (`flavor-flavor.zip`)

1. Log in to your WordPress Admin panel.
2. Go to **Appearance** → **Themes**.
3. Click the **Add New** button at the top, then click **Upload Theme**.
4. Choose the `flavor-flavor.zip` file you downloaded.
5. Click **Install Now**.
6. Once installed, click **Activate**.

> ⚠️ **IMPORTANT**: After activating the theme, go to **Settings → Permalinks**, make sure **Post name** is selected, and click **Save Changes**. This prevents `404 Not Found` errors on manhwa and chapter pages.

### 2️⃣ Installing the Scraper Plugin (`manhwa-scraper.zip`) _(Optional)_

1. Log in to your WordPress Admin panel.
2. Go to **Plugins** → **Add New**.
3. Click the **Upload Plugin** button at the top.
4. Choose the `manhwa-scraper.zip` file you downloaded.
5. Click **Install Now**.
6. Once installed, click **Activate**.
7. The Scraper menu will now appear in your WordPress sidebar.
